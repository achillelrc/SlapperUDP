# Slapper UDP ddos tool
Outil de spam par dénis de service écrit en C

# Utilisation
Besoin de la libc et de gcc

Compiler avec **gcc slapper.c -o slapper**

Puis **./slapper [ip] [port] [packet size (max:1000)] [time]**
